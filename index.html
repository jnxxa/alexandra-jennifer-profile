Problem: Users often struggle to manage and access up-to-date movie catalogs efficiently, especially when they need to track favorites without constant manual searching or reliance on a permanent internet connection.
Solution: Developed a mobile application that automates data ingestion from external REST APIs and processes raw JSON into a structured app model. Academic Computer Vision
Real Time Sign Language Detection System
Problem: There is a significant communication gap for ASL users due to the lack of accessible, real-time tools that can instantly translate complex hand gestures into digital text.
Solution: Engineered a Computer Vision system using MediaPipe to extract high-precision hand landmarks into structured numerical vectors. By applying image normalization and training SVM and Random Forest algorithms, the system achieves high-accuracy, real-time classification of ASL letters and numbers.
Python OpenCV MediaPipe Scikit-Learn

Academic Research Deep Learning
MobileNetV2 ~ Automated Waste Classification System
Problem: Inefficiency of manual waste sorting and dataset imbalance often reduce the accuracy of AI models in automatically recognizing types of waste.
Solution: Implementing the MobileNetV2 architecture with Transfer Learning techniques and systematic data augmentation. This research successfully improved the model's performance in classifying organic and inorganic waste in real-time.
